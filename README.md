# homebrew-tap

Homebrew tap for personal and experimental tools by [@yag13s](https://github.com/yag13s).

## Usage

```sh
brew tap yag13s/tap
```

## Formulas

|Formula|Description|
| :---: | :---: |
|goreach|Find unreached code paths in running Go services.|

### [goreach](https://github.com/yag13s/goreach)

```sh
brew install yag13s/tap/goreach
```
